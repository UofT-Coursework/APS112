# Engineering Strategies & Practice (ESP) Project Demo for Team049

Program to translate user inquiry regarding Sandford Fleming Library services into English and generate a response from GPT-4o.\
Many of the elements are primarily for show and not necessarily efficient. \
Built using Selenium, OpenAI, DeepL, LangDetect.

Features:
- Accept user inquiry and detect language
- Visually show inquiry being translated to English
  - Option to quickly translate short text for demo purposes.
- Visually show inquiry being inputed into ChatGPT (for show only)
- Recieve response from GPT-4o and return information & sources.

## Test Prototype

Actual program used to test for accuracy (AI Hallucination / Bias rates) as part of the ESP project.
